# ✅ QC Checklist (نمونه)

## IQC (ورودی)
- [ ] شماره پارت و مقدار صحیح
- [ ] ظاهر بدون آسیب/اکسید
- [ ] تست تصادفی عملکرد قطعات حساس

## IPQC (حین تولید)
- [ ] AOI پاس
- [ ] گشتاور پیچ‌ها مطابق SOP
- [ ] لحیم‌کاری تمیز، بدون بریج

## FQC/EOL (پایانی)
- [ ] بوت فریمور + نسخه ثبت‌شده
- [ ] WiFi/Zigbee/LoRa OK (RSSI ≥ معیار)
- [ ] سنسورها در رنج کالیبراسیون
- [ ] برچسب سریال/MAC/SKU
- [ ] ملزومات پک و دفترچه
